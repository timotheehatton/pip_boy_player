# Pip Boy 3000 player

A PipBoy player realised only in CSS and JavaScript

![](http://www.giphy.com/gifs/l3q2SJyVnAhICpObS)

### Features

  - play, mute, replay, fullscreen Button
  - Draggable progress bar
  - Dynamical time display
  - Volume setting by "+" and "-" button
  - Qualities selector
